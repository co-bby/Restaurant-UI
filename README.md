## Tech Stack

**Client:** React.js

**Server:** No Backend!!


## Run Locally

Clone the project

```bash
  git clone https://github.com/co-bby/Restaurant-UI
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Lessons Learned
